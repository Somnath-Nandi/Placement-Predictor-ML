**Placement Prediction Dataset**

This dataset is used for building a machine learning model to predict student placement status based on academic and cognitive attributes.

The dataset is commonly used in introductory ML classification problems, where the goal is to determine whether a student gets placed based on their CGPA and IQ.

**Dataset Columns**

1️⃣ **Index Column**

Column: Unnamed: 0

This is simply the dataset index.

It does not contribute to the model.

It can be removed during preprocessing.

2️⃣ **CGPA**

Column: cgpa

Represents the student's Cumulative Grade Point Average.

Higher CGPA generally indicates better academic performance.

3️⃣ **IQ**

Column: iq

Represents the IQ score of the student.

Used as an indicator of cognitive ability.

4️⃣ **Placement Status (Target Variable)
**
Column: placement

This is the target variable used in machine learning models.

1 - Student is Placed

0 - Student is not Placed
