**Student Placement Prediction using Machine Learning**

This project implements a Machine Learning classification model to predict whether a student will be placed based on their academic performance (CGPA) and IQ score.

The project demonstrates the complete ML pipeline, including data preprocessing, visualization, model training, and prediction.

**Technologies Used**

Python

NumPy

Pandas

Matplotlib

Scikit-learn

Mlxtend

These libraries are used for data analysis, visualization, and machine learning modeling.

**Machine Learning Workflow**

The project follows a standard machine learning pipeline:

1️⃣ Data Loading

The dataset is loaded using Pandas.

2️⃣ Data Preprocessing

Removing unnecessary columns (Column "Unnamed: 0")

Preparing feature (cgpa, iq) and target variables (placement)

3️⃣ Data Visualization

The dataset is visualized to understand relationships between variables using matplotlib.

4️⃣ Model Training

A Logistic Regression model is trained to learn patterns from the data.

5️⃣ Decision Boundary Visualization

Using visualization techniques to observe how the model separates classes using Pickle class.

6️⃣ Model Prediction

The trained model predicts whether a student will be placed.
